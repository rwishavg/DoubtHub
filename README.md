</div>

   ```
   git clone https://github.com/rwishavg/PassportJS-OAuth.git  
   ```   

   ```
   npm install
   ```
   ```
   Create: /utils/config.env
   ```
   Add the following in config.env:
   ```
   PORT=
   MONGO=
   NODE_ENV=
   G_CLIENT_ID=
   G_CLIENT_SECRET=
   ```
   ```
   npm run both
   ```
